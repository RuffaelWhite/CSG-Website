# CSG-Website

CyberShield Guardian (CSG) cybersecurity website project.

This project includes a professional website for ethical hacking tools, bug bounty services, and consulting.

## Features

- Responsive UI/UX design
- Tools download page
- Contact form
- Node.js Express backend server

## Setup

1. Install Node.js dependencies:

```
npm install
```

2. Run the server:

```
node server/index.js
```

3. Open your browser at http://localhost:3000
